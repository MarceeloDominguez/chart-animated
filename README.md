## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/chart-animated.git
```

Go to the project directory

```bash
  cd chart-animated
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```
